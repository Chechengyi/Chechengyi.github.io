# 最小堆

首先要理解 最小堆 是一颗完全二叉树，最小堆 对应的就有 最大堆，

  • 最小堆 是指堆中某个节点的值总是小于其子节点的值
  • 最大堆 是值堆中某个节点的值总是 大于其子节点的 值

要操作堆，首先要理解 ，若当前节点的下标为 i，则其父节点的下标为 i/2 ，其左子节点的下标为 i*2+1。

一般可以用一个数组来表示一个堆。而当放在数组中的时候，其下标是从0开始的，如图所示：

所以改变以下计算的公式：

  • 当前节点的父节点：Math.floor((i-1)/2)
  • 当前节点的左节点：2i+1
  • 当前节点的右节点: 2i+2
  • 当前堆的最后一个叶子节点：就是 最后一个节点的父节点，比如图中值为2就是最后一个叶子节点，count 为最小堆的个数，堆的最后一个叶子节点：Math.floor((count-2)/2

接下来的记录全部以最小堆作为记录

总结来说，在初始化最小堆的过程中就是，首先找到最后一个叶子节点，然后让它与自己的子节点比较一直向下找直到它比自己的子节点都要小跳出这个过程，或者是往后已经没有叶子节点了。称这个过程为 shiftDown 操作，然后编写代码，首先声明一些保存堆的变量，编写一个初始化堆的方法和 shiftDown 方法：

var minHeap = [];
var minHeapCount = 0;

// 初始化最小堆
function heapfy(arr){
    minHeap = arr;
    minHeapCount = minHeap.length;
    var lastLeafChild = Math.floor((minHeapCount-2)/2);
    while ( lastLeafChild>=0 ) {
        shiftDown(lastLeafChild);
	    lastLeafChild --;   
    }
}

function shiftDown(index){
    while( (index+1)*2-1 <= minHeapCount ){
		var childIndex = (index+1)*2-1;
		if ( 
            (childIndex + 1) <= minHeapCount 
            && minHeap[childIndex] > minHeap[childIndex+1] 
        ) 
        {
			childIndex += 1
		}
		if ( minHeap[index] > minHeap[childIndex] ) {
			swap(arr, index, childIndex)
			index = childIndex
		} else {
			break;
		}
	}
}

function swap(arr, j, i) {
	var mid = arr[j];
	arr[j] = arr[i]
	arr[i] = mid
}

这样就完成了初始化堆的操作。

# 2 堆排序 || 优先队列出列

首先要明白一点，堆是可以用来做优先队列的，如果最小就代表最优先，那么堆的根节点肯定就是最优先的。

试想，如果我们要做从小到大的排序或者是做优先队列的出队操作，根节点就是最小的，如果我们每次都取出根节点，然后在重新生成一次堆，直到将堆里面的节点全部取完不就可以实现堆排序或者优先队列了吗？事实上不用每次都重新去生成堆这么麻烦。

堆的根节点被拿走了之后首先是要继续去维持堆这么一个结构，将最后一个节点放到根节点的位置，然后删掉最后一个节点。在对根节点做 shiftDown 操作就可以了。来编写一个出队也就是拿出根节点的方法：

function shift(){
	var mid = minHeap[0];
	minHeap[0] = minHeap[minHeapCount-1]
	minHeapCount--
	minHeap.length = minHeapCount;
	shiftDown(0)
	return mid
}

堆排序就返回的调用 shift 直到 minHeapCount 为 0 就行了。
