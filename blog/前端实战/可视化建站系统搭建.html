# 可视化建站系统搭建

最终页面的呈现通过一个统一的渲染器输出，生成页面的时候数据落库，生成页面唯一的标识id。在编辑页面的时候，添加组件。数据库有对应的组件表和页面表，页面和组件之间的关系为1对多，组件表里保存了组件的相关信息，组件id，组件存储的路径，组件 JSON schema等信息。 页面组件表 则表示了一个页面引用了哪些组件，对于引用的这个组件传入的属性是什么，数据是什么，都以JSON字符串的形式存储在页面组件表中。

# 服务端渲染器

当组件打包好存储到服务端、页面数据落库到数据库后，接下来就是要生成页面。

与普通开发的流程不一样，页面没有走开发，数据都直接存储到了数据库里面，所以即使使用了react这样的框架，在访问页面链接的时候，服务端应该直接返回生成好了的html代码。说白了，就是要在这里做服务端渲染。
